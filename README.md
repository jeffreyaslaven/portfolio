# Jeffrey Slaven — Portfolio

Personal portfolio website built with React and Vite.

## What's inside

- **Hero** — intro section with name and title
- **About** — background summary and skills
- **Projects** — personal projects linking out to GitHub
- **Experience** — work history and education timeline
- **Contact** — contact form powered by Formspree, plus links to email, LinkedIn, and GitHub

## Tech stack

- [React](https://react.dev) — UI framework
- [Vite](https://vitejs.dev) — build tool and dev server
- [Formspree](https://formspree.io) — contact form email delivery
- CSS custom properties for light/dark theming (respects system preference)

## Running locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run build
```
